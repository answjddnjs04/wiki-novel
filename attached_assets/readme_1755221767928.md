# README — 위키형 참여소설 플랫폼 설계도


## 주요 API 설계
### 1. 인증
- POST `/auth/signup`
- POST `/auth/login`

### 2. 소설
- GET `/novels`
- GET `/novels/:id`
- POST `/novels` (신규 작성)
- PATCH `/novels/:id/block` (블록 수정 제안)

### 3. 투표
- POST `/votes/:blockId`
- GET `/votes/:blockId`

---

## DB 스키마 요약
**User**
- username
- email
- password
- contributionScore
- titles

**Novel**
- title
- genre
- blocks: [BlockID]

**Block**
- content
- orderIndex
- contributors: [{ userId, charCount }]
- versionHistory: [ { version, content, date } ]

**Vote**
- blockId
- votes: [{ userId, weight, voteType }]
